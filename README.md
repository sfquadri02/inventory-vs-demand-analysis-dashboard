# inventory-vs-demand-analysis-dashboard
Power BI dashboard analyzing inventory vs demand performance, stock health, and key supply chain KPIs

## Project Overview
This project focuses on analyzing inventory levels against customer demand
to identify stock gaps, overstock situations, and demand trends.
The dashboard helps support data-driven inventory planning and decision-making.

## Dataset
- Source: Kaggle – Inventory Management Dataset by Hetul Parmar
- Link: https://www.kaggle.com/datasets/hetulparmar/inventory-management-dataset
- Format: CSV / Excel
- Description: Dataset contains product-level inventory, demand, and sales-related attributes used to analyze stock health and demand patterns

## Business Questions Addressed
- Is current inventory sufficient to meet customer demand?
- Which product categories face overstock or stockout risks?
- How does demand change over time?
- Which products have the highest demand and inventory gaps?

## Tools and Technologies
- Power BI Desktop
- Microsoft Excel
- DAX
- Data Modeling
- Exploratory Data Analysis (EDA)

## Key Metrics
- Total Demand
- Total Inventory
- Inventory Gap
- Inventory Turnover
- Revenue

## Dashboard Pages
- Inventory Overview: High-level inventory and demand KPIs with stock status
- Product-Level Analysis: Product-wise demand, inventory gap, turnover, and revenue
- Demand Trend Analysis: Year–month demand trends

## Key Insights
- Identified product categories with overstock and stockout risks
- Highlighted inventory gaps impacting demand fulfillment
- Analyzed demand trends to support forecasting and planning
